Vizualizace

Množina

Množina je datová struktura, která ukládá jedinečné prvky bez konkrétního pořadí. Představit si ji můžete jako soubor různých předmětů na stole, kde každý předmět může být přítomný pouze jednou bez ohledu na to, jak je uspořádán.

Operace

  • Add (přidání) - přidá prvek do množiny, pokud tam již není
  • Remove (odebrání) - odebere prvek z množiny
  • Contains (obsahuje) - zkontroluje, zda množina obsahuje daný prvek

Časová komplexita

  • Přidání: O(1)
  • Odebrání: O(1)
  • Obsahuje: O(1)

Kód

JavaScript

                            
class CustomSet {
  constructor() {
    this.items = {};
  }
  
  add(element) {
    if (!this.contains(element)) {
      this.items[element] = element;
      return true;
    }
    return false;
  }
  
  remove(element) {
    if (this.contains(element)) {
      delete this.items[element];
      return true;
    }
    return false;
  }
  
  contains(element) {
    return Object.prototype.hasOwnProperty.call(this.items, element);
  }
  
  values() {
    return Object.values(this.items);
  }
  
  size() {
    return Object.keys(this.items).length;
  }
  
}
